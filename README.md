# OnlineCalc

Demo:
![hippo](https://github.com/h1kix/onlinecalc/blob/main/demo/2026-03-20%2005-50-09.gif)

Небольшой онлайн‑калькулятор на чистых `HTML + CSS + JS`.

- Приоритет операций: `*` / `/` выше `+` / `-`
- Скобки: `(` `)`
- Дробные числа без типичных ошибок `float` (вычисление через рациональные числа на `BigInt`)
- История операций (интерактивная) + сохранение в `localStorage`

## Структура проекта

- `index.html` — разметка (калькулятор + история)
- `styles.css` — стили
- `app.js` — логика (парсер выражений, вычисления, UI, история)

## Локальный запуск

### Вариант 1 (самый простой)

Открой `index.html` в браузере двойным кликом.

### Вариант 2 (через простой статический сервер)

Если удобнее гонять как сайт:

1. Установи любой static server (пример для Node.js):

```bash
npx serve
```

2. Открой адрес, который он напишет в консоли.

## Как пользоваться

### Кнопки

- `CE` — очистить последний ввод числа ("clear entry")
- `⌫` — удалить 1 символ
- `=` — посчитать
- Кнопка `⌫` в шапке — сбросить всё

### Клавиатура

- `Enter` или `=` — посчитать
- `Backspace` — удалить 1 символ
- `Esc` — сбросить всё
- `,` — вставляется как `.` (удобно под RU раскладку)

### Поддерживаемые выражения

- Операторы: `+`, `-`, `*`, `/`
- Скобки: `(`, `)`
- Дроби: например `12.34 + 0.06`

Примеры:

- `2+2*2` -> `6`
- `(2+2)*2` -> `8`
- `0.1+0.2` -> `0.3`

## История операций

После нажатия `=` операция попадает в историю.

- Кнопка `↩` — подставляет выражение обратно в калькулятор
- Кнопка `✕` — удаляет запись из истории
- `Очистить` — чистит всю историю

История хранится в браузере в `localStorage` по ключу:

- `onlinecalc.history.v1`
